第２週ミニ課題

/todo に GET のリクエストを送った時に，返ってくるレスポンス（ToDoリストの内容）をテキストとして pre タグを用いて記述すること．見やすくなるように適宜改行を入れてもかまわない．

【ヒント】ブラウザのWeb開発ツールを使えばレスポンスの内容が調べられる．

    
    [
      {
        "id":"0",
        "task":"実行済のタスク",
        "completed":true
      },
      {
        "id":"1",
        "task":"これから実行するタスク",
        "completed":false
      }
    ]
